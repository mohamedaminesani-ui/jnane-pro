# Jnane Pro — Design Preview

Prototype de design de l'application **Jnane Pro** (services de jardinage — Maroc / MENA), généré avec Claude Design.

## 🌿 Voir le design

Ouvre simplement `index.html` dans un navigateur, ou visite la version live sur GitHub Pages.

## 📱 Écrans inclus

- Onboarding (3 slides)
- Home
- Liste des services
- Détail service
- Demande de devis (Quote form)
- Suivi d'intervention (Tracking)

## 🛠️ Stack

- React 18 (via CDN)
- Babel standalone (compile JSX en navigateur — pratique pour prototype)
- DM Sans + Instrument Serif (Google Fonts)

## 🎨 Palette

- Vert profond `#1E5B3A`
- Vert feuille `#7CB342`
- Sable `#E8DCC4`
- Crème `#FBF7EE`
